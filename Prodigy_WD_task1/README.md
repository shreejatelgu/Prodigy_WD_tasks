# Animated Web Page with AOS and Feather Icons

This project is a simple, responsive web page that uses **AOS (Animate On Scroll)** for smooth scroll-based animations and **Feather Icons** for lightweight, customizable icons.

---

## Features

- **Scroll Animations:** Elements animate as they come into view using [AOS](https://michalsnik.github.io/aos/).  
- **Feather Icons Integration:** Vector icons replaced automatically using `feather.replace()`.  
- **Custom Styling:** Fully styled with CSS for layout and responsiveness.  
- **Lightweight and Easy to Use:** Only a few external libraries needed.

---

## Technologies Used

- **HTML5** – Structure of the webpage  
- **CSS3** – Styling and animations  
- **JavaScript** – To initialize AOS and Feather icons  
- **AOS** – Animate on scroll library  
- **Feather Icons** – Icon library
